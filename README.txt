Die TEQnners Slideshow (tq_slideshow) ist eine speziell für TYPO3 entwickelte Bilder Slideshow.
Sie liefert Webentwickler einen Werkzeugkasten um mehrere Bilder innerhalb eines Conatiner auf Ihrer Webseite mit
modernen Übergangen darzustellen.

TEQneers Slideshow - Eine zukunftsichere TYPO3-Extension

Die TEQneers Slideshow basiert clientseitig auf dem Javascript Framework jQuery und dem jQuery Plgin jQuery Cycle.
Serverseitig basiert die TEQneers Slideshow auf dem MVC-Framework Extbase und der Template-Engine Fluid.
Durch die Unterstützung von Extbase ist die TEQneers Slideshow eine zukunftssichere TYPO3-Extensions und
dank der vollständigen Unterstützung der neuen Template-Engine Fluid ist der Gestaltung Ihres individuellen Themes
keine Grenzen gesetzt. Ein Beispiel-Template liefern wir im Basispacket mit damit die Anpassungen keine Probleme bereiten.


Individuelle Übergänge

Durch die Verwendung von jQury Cycle basiert die TEQneers Slideshow auf ein jQuery Plugin das bereits heute auf
vielen Webseiten Verwendung findet. Dadurch wird eine hohe Qualität und Zuverlässigkeit geboten.
Im Basispacket liefert jQuery Cycle über 25 moderne und elegante Übergänge zwischen 2 Bildern mit.
Für Individualisten ist es Ihnen jederzeit möglich auch Ihre eigenen Übergänge zu definieren.


TEQneers Slideshow - Mehr als nur eine Slideshow

Die TEQneers Slideshow bietet neben der Darstellung von Bildern und den eindrucksvollen Effekte auch die Möglichkeit
Thumbnails oder eine Toolbar darszustellen. Das ermöglicht den Webseiten Besucher eine einfach Navigation durch Ihre
Bilder.

Die TEQneers Slideshow unterstützt die vollständige Funktionalität einer Lightbox. Dadurch können Beschreibungstexte oder
Bilder in einer Lightbox dargestellt werden.


Features:

- Unendlich viele Bilder innerhalb eines Containers
- Mehrere Slideshows auf einer Seite
- Alternative Anzeige von Text oder externen Inhalt in einer Lightbox
- Unterstützt Key-Events
- Unterstützung von über 25 moderne Übergänge
- Definierung von eigenen Übergänge
- Anzeigen von Thumbnails
- Definierung von Alternativen Thumbnails zu einem Bild
- Anzeigen einer Toolbar




The TEQneers Slideshow (tq_slideshow) is a special for TYPO3 developed Image Slideshow.
